# Theralink

Sistema de gestão para psicólogos com prontuários, videochamada, chat e pagamentos.

## Stack
- Frontend: HTML + CSS + JS
- Backend: Node.js + Express
- Banco: Supabase (PostgreSQL)
- Videochamada: Daily.co
- Pagamentos: Stripe
- Chat: Socket.io

## Setup

```bash
cd backend
npm install
cp .env.example .env
# preencha as variáveis no .env
npm run dev
```
